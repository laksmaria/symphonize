This project is a data generation library for Orchestrate.io services. It can be used locally or as a tool to create randomly created data based on configuration into the Orchestrate.io services for testing, experimentation or however or whatever you may need data in Orchestrate.io for. For more information on the library ping me @adron or on Twitter @ the same.

Symphonize

Mission:

The purpose of this library is to provide a configuration based way to generate data and get that data into a data source, initially that data source is Orchestrate.io.

Dependencies:

In this project we've used several projects, all available via NPM. These projects include: orchestrate.js and chance.js.
